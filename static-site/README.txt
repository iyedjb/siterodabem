RODA BEM TURISMO - Static Site
================================

This folder contains everything you need to host on Hostinger!

FILES:
- index.html    (main page)
- styles.css    (all styling)
- favicon.png   (browser tab icon)
- images/       (all destination photos)

HOW TO UPLOAD TO HOSTINGER:
1. Log in to your Hostinger account
2. Go to File Manager (or use FTP)
3. Navigate to public_html folder
4. Upload ALL files from this folder:
   - index.html
   - styles.css  
   - favicon.png
   - images folder (with all photos inside)
5. Done! Your site is live!
